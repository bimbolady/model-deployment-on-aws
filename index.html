Welcome to the landing page for the Bosch Model.

To use the model for predictions, do the following:

Correct Endpoint: Google Postman on your preferred browser and add /predict to this website's url.

To update your query in Postman, please follow these steps:

  • Open Postman and add the current url but with the /predict endpoint. The endpoint is a post request which means you cannot access it on the browser.
  • Ensure the HTTP method is post.
  • Verify that your request headers, parameters, and body are correctly configured. The body sample is shown in the github link.
  • Click Send to make the request.
